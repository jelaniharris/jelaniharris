# Hello there fellow coder. I'm Jelani Harris 👋

## About Me:

I am a passionate Full Stack Web Developer with 14 years of experience. I'm a fiction writer, and video game enthusiast, and indie game developer in my spare time.

## 🔭 Currently Working on

Currently, I'm a Software Engineer Consultant at the [Minnesota Star Tribune](https://startribune.com). I have a background in working on various applications in Finance, SaaS, eCommerce, Internal Tools, Education, and Health Care.

## 🌱 Currently Learning

I'm working on getting better at NoSQL databases after having decades of relational database experience. I've dabbled in MongoDB and DynamoDB.

I'm also trying to improve my mobile skills by learning React Native (again).

After taking a year off, I've been trying to get back into Unreal Engine.

## Check out My Work / Personal Projects

- [Personal Website](https://jelaniharris.com) - My personal website where I usually post about issues and errors I run into to help someone else. Made in Gatsby and Bulma.
- [Tower of Greed](https://tower-of-greed.vercel.app) - A 3d procedural-generated dungeon crawling game where you must get as high as possible and escape the dungeon without dying. Made in React, Next.JS, and Three.JS
